# Box Breathing Lite

Бесплатная веб-страница для квадратного дыхания. Работает полностью в браузере, сервер не нужен.

## Возможности

- Фиксированный режим (например 4-4-4-4)
- Прогрессивный режим (4 → 5 → … → 20)
- Лимит по проходам или по времени
- Визуальный квадрат с анимацией
- Пауза / продолжить / стоп
- Сохранение настроек в `localStorage`

## Запуск локально

```bash
cd web-lite
npm install
npm run dev
```

Открой http://localhost:5173

## Сборка

```bash
npm run build
```

Результат в папке `dist/` — можно выложить на GitHub Pages, Cloudflare Pages или Netlify.

## Тесты

```bash
npm test
```

## Деплой на GitHub Pages

1. Собери проект: `npm run build`
2. Опубликуй содержимое `dist/` (через GitHub Actions или ветку `gh-pages`)
3. В настройках репозитория включи Pages

`vite.config.ts` уже использует `base: './'` для корректных путей на статическом хостинге.
